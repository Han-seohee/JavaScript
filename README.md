# JavaScript 자바스크립트 이론
### :star2:변수
```
let value = 1; 
console.log(value);
//특정 이름에 특정 값을 넣는것 (선언)

value = 2;
console.log(value);
// 값을 바꿀 수 있음

let value = 2; //error - 한 번 선언 했으면 똑같은 이름으로 선언 불가
```

### :star2:상수
```
const a = 1; //한번 값을 설정하고 나면 바꾸지 못함
a = 2; //error
const a = 2; //error
```
### :star2:데이터 타입

